API: URL Shortener Microservice

Create a shortened URL, either by submitting the URL in the form below or by visiting the API endpoint at:

GET https://damp-citadel-44118.herokuapp.com/api/shorturl/new/

Example:
https://damp-citadel-44118.herokuapp.com/api/shorturl/new/https://www.google.com
Response:
{"original_url":"www.google.com","short_url":"1111"}

Example:
POST https://damp-citadel-44118.herokuapp.com/api/shorturl/new - https://www.google.com

Example Usage:

By visiting the output short URL: https://damp-citadel-44118.herokuapp.com/api/shorturl/8115

You will be redirected to: http://www.freecodecamp.com
