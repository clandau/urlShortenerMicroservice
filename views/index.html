API: URL Shortener Microservice

Create a shortened URL, either by submitting the URL in the form below or by visiting the API endpoint at:

GET [project-url]/api/shorturl/new/

Example:
[project-url]/api/shorturl/new/https://www.google.com
Response:
{"original_url":"www.google.com","short_url":"1111"}

Example:
POST [project-url]/api/shorturl/new - https://www.google.com

Example Usage:

By visiting the output short URL: [project-url]/api/shorturl/4121

You will be redirected to: http://www.freecodecamp.com
